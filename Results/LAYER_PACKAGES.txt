Layer Packages
==============
The following ArcGIS layer packages (.lpkx) exist in this directory but are
excluded from version control due to file size.  Store and share them via
network drive, external storage, or ArcGIS Online.

  TNC_Appalachian_Indices.lpkx        (~11 GB)
    Monthly max composites for NDVI, NDMI, EVI — Sentinel-2
    Covers north (GWNF) and south (Smoky) AOIs, 2017–present

  TNC_Appalachian_Indices_Monthly.lpkx  (~87 GB)
    Full monthly layer stack — Sentinel-2
    Covers north (GWNF) and south (Smoky) AOIs, 2017–present

These packages were generated from the GWNF_cache_3_4 / Smoky_cache_3_4
index caches using the compositing scripts in Analysis/.
A rebuilt cache with corrected spatial masking (AOI polygon instead of
bounding box) and cloud-only SCL exclusion is being built as
GWNF_cache_3_24 / Smoky_cache_3_24.  Layer packages will be regenerated
from that cache once complete.
